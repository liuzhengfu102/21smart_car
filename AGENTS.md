## 技术栈

- **语言**：C++
- **目标架构**：ARM64（Orange Pi 5）
- **编译**：需交叉编译或在板卡上本地编译（使用 `aarch64-linux-gnu-g++` 或板载 g++）

## 开发工作流

1. 本地编写 C++ 代码
2. 交叉编译为 ARM64 可执行文件（或在 Orange Pi 5 上直接编译）
3. 通过 SCP/SFTP 将可执行文件传输到板卡
4. SSH 连接板卡运行测试

### 传输文件到板卡

```bash
scp <本地文件> orangepi5@192.168.3.57:<远程路径>
# 密码: orangepi
```

### 在板卡上编译（若选择本地编译）

```bash
ssh orangepi5@192.168.3.57 "cd <项目目录> && g++ -std=c++17 -O2 *.cpp -o <输出文件>"
```

## 注意

- Orange Pi 5 为 ARM 架构，本地 x86 编译的二进制无法直接在板卡上运行
- 需确认板卡上已安装必要的运行时库（如 OpenCV、TensorRT 等 AI 相关库，取决于赛题需求）
- 板卡 IP 可能变化，连接失败时需确认 IP 地址
- **禁止在代码或提交中硬编码密码等敏感信息**——以上密码仅用于 SSH/SCP 运维操作
